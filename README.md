# `stress.m` Stress-strain Plotter

This utility will graph atress-strain plot with GNU Octave.

# Usage

Example:

```
$ octave-cli stress.m examples/St1594.csv
St1594
E = 159.5514e+009
y =

     4.0547e-03   327.8306e+00

UTS = 497.0183e+000
Ur = 336.7970e+003
Ut = 176.4531e+006
UrA = 976.8397e+003
Ury = 664.6275e+003
```
